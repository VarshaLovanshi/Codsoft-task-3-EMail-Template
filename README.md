# Codsoft task-3 EMail Template

OVERVIEW:

This project is a promotional email template designed in Figma for Feasto, a food delivery application. The template advertises a 20% discount offer using a clean, modern, and visually appealing layout that encourages users to place an order. The design follows UI/UX principles to ensure readability, strong visual hierarchy, and an engaging user experience.

FEATURES:
1. Promotional banner highlighting the 20% discount
2. Attractive food-themed visuals
3. Clear and prominent Call-to-Action (CTA) button
4. Consistent color palette and typography
5. Mobile-friendly email layout
6. Brand-focused design for Feasto

DESIGN GOALS:
1. Capture users' attention instantly
2. Increase engagement through promotional content
3. Present information in a clear and organized manner
4. Maintain visual consistency with the Feasto brand
5. Deliver a professional and user-friendly email experience

TOOLS USED:
Figma

LEARNING OUTCOMES:
1. Designed a marketing email using UI design principles
2. Improved skills in layout, typography, spacing, and color selection
3. Learned how to create promotional content that balances aesthetics with usability
4. Gained experience designing responsive email interfaces suitable for mobile users

This project was completed as Task 3 of my UI/UX Design Internship at CodSoft.
